# Contributing

Thanks for helping improve this awesome list.

## Submission Guidelines

- Suggest tools that are directly relevant to fitness trainers, personal coaches, strength coaches, or gym professionals.
- Use real, working product URLs. Official product pages are preferred.
- Include a short explanation of why the tool matters for fitness trainers.
- Keep descriptions factual, specific, and non-promotional.
- Prefer tools that are active, current, and clearly useful in 2026.
- Check for duplicates before submitting a new entry.
- If pricing is public, include it. If pricing is not public, note that it is custom or quote-based.

## Quality Standard

To keep the list useful, new entries should ideally include:

- Product name
- Official URL
- Clear category fit
- Unique description
- Category tags
- Pricing summary
- Best-for recommendation

## Pull Request Process

1. Fork the repository.
2. Create a focused branch for your change.
3. Update `README.md` with your addition or improvement.
4. If you add or remove a major section, update the table of contents.
5. Submit a pull request with a short explanation and relevant sources.

## What Not to Add

- Dead links
- Duplicate tools
- Generic AI apps with no clear trainer use case
- Low-quality affiliate-style descriptions
- Unverified pricing claims

## Style Notes

- Keep formatting consistent with the existing list.
- Use concise, readable English.
- Prefer neutral editorial tone over marketing copy.

By contributing, you agree that your submissions may be edited for clarity, consistency, and quality.
